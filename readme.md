Javaで実装しました．

DFSで探索爆発が起きそうだったので異常なまでに制限してます．
---
1. 盤上の駒のうちランダムに１つを選択し，縦横３マス幅で相手駒を探しそこの付近に移動させます．
2. 1が達成できない場合，ランダムに１つ選択して１マス動かします．

実装していないはさみ将棋のルールがあります
